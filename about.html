About Recipe Explorer

Recipe Explorer is a web application created by Andrea Nicholson, Sarah Werner, Brigit Williams, and Michael Simmons for The Coding Bootcamp at UNC Chapel Hill. Recipe Explorer is powered by the MealDB API (https://www.themealdb.com/api.php) and the Edamam Nutrition Analysis API (https://developer.edamam.com/edamam-nutrition-api).
